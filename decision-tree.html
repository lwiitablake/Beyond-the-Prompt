INTERACTIVE DECISION TREE

Find the right next step

Use this page when you are not sure where to begin.

Open the branch closest to your task. Each branch asks a few questions and points you toward a next step.

Before you start

What should you do? Start with the task, not the tool.

Why is this better for learners? Starting with the task keeps AI use focused on learning, access, privacy, and support.

What are you trying to do?

Open one branch. Keep the rest closed unless you need them.

Branch 1 I want to use AI for a course task Should AI be involved at all?

Decision path

Question: Does the task involve private learner information?

If yes: Do not use a public AI tool.

If no: Continue.

Question: Does the task affect grades, eligibility, accommodations, discipline, or access?

If yes: Keep human judgment central and check institutional policy.

If no: AI may be appropriate for brainstorming, drafting, organizing, or revision.

Next step

Open the Decision Cards and use “Should AI be used here?”

Branch 2 I want to upload something Is it safe to share with an AI tool?

Decision path

Question: Does it include names, grades, IDs, messages, accommodations, or learner work?

If yes: Stop. Do not upload it to a public AI tool.

If no: Continue.

Question: Does it include copyrighted or licensed material?

If yes: Check the license or use a short excerpt only if allowed.

If no: You may continue with human review.

Next step

Use the “Can I upload this?” Decision Card.

Branch 3 I want to improve course materials Will this better support learners?

Decision path

Question: Can learners tell what to do first?

If no: revise the structure and directions before adding more content.

If yes: continue.

Question: Are links, files, images, tables, and media accessible?

If no: fix access barriers before publishing.

If yes: continue.

Question: Does this support the learning goal?

If no: revise or remove it.

If yes: it may be ready for final review.

Next step

Use the “Is this ready for learners?” Decision Card.

Branch 4 I want learners to use AI Does AI support or replace the learning?

Decision path

Question: What skill are learners supposed to practice?

If AI completes that skill for them: restrict AI or redesign the task.

If AI supports brainstorming, revision, practice, or reflection: it may be appropriate.

Question: Are expectations clear?

If no: explain what AI use is allowed, what must be disclosed, and what learners must do themselves.

If yes: continue.

Next step

Use the “Should learners use AI for this?” Decision Card.
